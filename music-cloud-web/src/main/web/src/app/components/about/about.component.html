Music-cloud Дипломная работа Кулыгина Виктора

При реализации приложения music-cloud используется следующий стек технологий: Java 8, Spring Boot 2, Liquibase, Lombok, Hibernate 4, система сборки - Maven 3, СУБД - Posgresql 9.6(prod) и MySql(dev). Фреймворк для разработки клиентской части веб-приложений Angular 6. Взаимодействие между серверной и клиентской частью происходит по протоколу REST. В качестве системы контроля версий используется Git. Также приложение имеет встроенную рекомендательную систему, построенную на принципе коллаборативной фильтрации
